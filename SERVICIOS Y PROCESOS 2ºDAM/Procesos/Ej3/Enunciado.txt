Crea un programa Java que visualice 5 veces la cadena que se le envía desde los 
argumentos de main() y finalice el programa con System.exit(0). Si no se le envía 
ninguna cadena que muestre un mensaje indicándolo y que finalice el programa con 
System.exit(1).
A continuación, crea un segundo programa Java que introduzca por teclado una 
cadena y ejecute el programa anterior para visualizar 5 veces esa cadena.

Create a Java program that displays 5 times the string sent to it from the
arguments to main() and end the program with System.exit(0). If you are not sent
no string displaying a message indicating so and ending the program with
System.exit(1).
Next, create a second Java program that will type in a
string and run the above program to display that string 5 times.
